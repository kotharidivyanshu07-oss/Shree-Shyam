# Shree Shyam Travel and Tour 🚖

A simple and responsive **cab booking and travel service website** built to help customers easily book rides and explore travel services online.

## 🌐 Live Demo

👉 https://shreeshyamtravelandtour.vercel.app/

---

## 📌 Features

* 🚕 Cab booking service
* 📱 Responsive design (works on mobile and desktop)
* 📍 Travel and tour service information
* 📞 Easy contact options for customers
* ⚡ Fast loading website deployed on **Vercel**

---

## 🛠️ Technologies Used

* HTML
* CSS
* JavaScript
* Vercel (for deployment)

## 🚀 Deployment

The project is deployed using **Vercel** for fast and reliable hosting.

Steps to deploy:

1. Push the project to GitHub.
2. Connect the repository with **Vercel**.
3. Deploy automatically with continuous updates.
